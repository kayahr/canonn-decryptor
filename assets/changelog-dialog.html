Changelog

Version 1.4.0 (2025-10-07)

  • Added support for custom space characters in Morse encoder and decoder.
  • Updated ngram database to improve automatic keyword cracking.
  • Smaller bugfixes and improvements in Polybius and Caesar converters.
  • Rename Decrypt button to Convert.
  • Some CSS improvements.

Version 1.3.0 (2025-10-03)

  • Implemented Affine cipher support.
  • Implemented Polybius Square support.

Version 1.2.0 (2025-09-02)

  • Added number encoder.
  • Fixed negative number handling in number decoder.

Version 1.1.2 (2025-09-01)

  • Fix broken icons.

Version 1.1.0 (2025-09-01)

  • Modernized this 8 year old project to newest Angular, TypeScript and all the bells and whistles.
  • Added a Clear button.
  • Changed toast colors. Now distinguishes between info and warning messages.

Version 1.0.0 (2017-12-03)

  • Update N-gram database (used for automatic caesar/keyword cracking) with current Canonn lore, codex and news.
  • Improve editing performance by de-bouncing the URL update.
  • Implemented Base64 encoding support.
  • Implemented One-time pad cipher support.
  • Implemented this changelog dialog you are currently looking at.

Version 0.0.1 (2017-10-10)
